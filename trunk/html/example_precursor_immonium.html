If the precursor ion is found in the scan it is labeled unless labelPrecursorPeak is set to false in the options. 
labelPrecursorPeak: false
precursorMz and charge are required in the options for labeling the precursor peak.

Immonium ions checkbox is checked by default. To leave it unchecked when the page loads set labelImmoniumIons to false. 
labelImmoniumIons: false

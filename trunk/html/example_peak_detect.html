Lorikeet Plugin Example

The "Peak Detect" option, checked by default, selects only those peaks that are above a noise threshold for labeling with fragment ions.
For each peak, Lorikeet determines whether that peak is significant by calculating an average & standard deviation using a +/- 50 Da window around that peak. If the intensity of the peak is 2 standard deviations above the average intensity, then it is eligible to be labeled with a fragment ion.
In addition, The top 50 most intense peaks are always included for labeling with fragment ions.

To turn off peak detection set the "peakDetect" option to false. 
        peakDetect: false

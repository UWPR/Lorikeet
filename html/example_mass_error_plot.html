Lorikeet Plugin Example

To display a plot showing the mass errors for annotated peaks, set 'showMassErrorPlot' to true (default is false). Default unit for mass error is Daltons ('Da'). Set massErrorPlot_default_unit to 'ppm' to display mass errors in ppm when Lorikeet initializes. Users can toggle between 'Da' and 'ppm' by clicking on a link in the mass error plot. 
showMassErrorPlot: true
massErrorPlotDefaultUnit: 'Da'
